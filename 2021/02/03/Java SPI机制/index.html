什么是Java SPI

参照百度百科的SPI的解释：百度百科

定义

SPI(Service Provider Interface)：是一个内置的java标准，允许不同的开发去实现某个特定的服务，服务提供一个标准的接口及用于提供某种特定的服务的接口

性质

SPI是Java 1.5新添加的一个内置标准，允许不同的开发者去实现某个特定的服务。一个Service就是一个接口或抽象类，而Service Provider是这个Service的一个特定实现类 。
